##  Problem Statements
 
1.  **Greatest of 2:** Find the greatest number from 2 user inputs.
2.  **Greatest of 3:** Find the greatest number from 3 user inputs.
3.  **Greatest of 4:** Find the greatest number from 4 user inputs.
4.  **Natural Sum:** Find the sum of first N natural numbers.
5.  **Factorial:** Find the factorial of a given number N.
6.  **Alternating Series:** Print the pattern: `1, -2, 3, -4, 5...`
7.  **Factorial Series:** Print the pattern: `1!, -3!, 5!, -7!...`
8.  **Number Series:** Print the pattern: `1, 12, 123, 1234...`
9.  **Palindrome Series:** Print the pattern: `1, 121, 12321, 1234321`
10. **Incrementing Palindrome:** Print the pattern: `1, 232, 34543, 4567654... n`
11. **Incrementing Palindrome With %10:** Print the pattern: `1, 232, 34543... 67890109876`
12. **Odd-Fibonacci Sequence:** Print the pattern: `1113253759...`
13. **Series(Sine):** Solve for: x - x^3/3! + x^5/5! - x^7/7! + ... upto n.
