# Assignment: HTML & CSS Login/Registration Page

A responsive, login/registration UI built with plain HTML and CSS.

## Files

- `index.html` – Markup for the split layout and toggleable Login/Register forms (radio-tab based).
- `style.css` – Styling for layout, cards, toggles, and responsive breakpoints.

## Features:

- **Dual Functionality:** Toggle between Login/Registration forms using tabs.
- **Modern UI Design:**
  - Split-screen layout (70% left side with branding, 30% right side with form)
  - Professional color scheme with blue accent colors
- **Responsive Design:** Mobile-friendly layout that adapts to different screen sizes
- **Form Elements:**
  - Login form: Username and Password fields with "Forgot Password?" link
  - Registration form: Username, Email, and Password fields
- **Visual Elements:**
  - Company logo display
  - Welcome message and description
  - Clean, modern interface

## How to Run

- Open `index.html` directly in a browser—no dependencies or server needed.
- Resize the window to observe the responsive stacking behavior.
